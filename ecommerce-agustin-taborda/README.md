## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Technologies

- **Framework**: NestJS
- **ORM**: TypeORM
- Database**: PostgreSQL
- Server**: Express
- Language**: Node.js and TypeScript
- Image Storage**: Cloudinary

## Installation
